# Getting Started

```bash
git clone git@github.com:rberruezo/manual-order.git
cd manual-order
npm install
npm start
```

# Steps to run

#### Compile
```bash
npm start
```
#### Start server
```bash
python -m SimpleHTTPServer 8000
```
#### See it
Open a browser and go to `http://localhost:8000/index.html`
